🎮 Proyecto Multi-Juegos: El Ahorcado & Piedra, Papel o Tijera 

Bienvenido a nuestro repositorio. Este proyecto es el resultado de un trabajo colaborativo donde hemos desarrollado dos juegos clásicos utilizando lenguaje Python, enfocándonos en la lógica de programación y el trabajo en equipo.

🕹️ Los Juegos 

1. El Ahorcado 😵 

El clásico juego de adivinar palabras antes de que el dibujo se complete. 

Características: 

Base de datos de palabras aleatorias. 

Control de intentos fallidos. 

Visualización del progreso de la palabra. 

2. Piedra, Papel o Tijera 👊✋✌️ 

Un duelo de estrategia contra la computadora. 

Características: 

Validación de entradas del usuario. 

Generación de jugada aleatoria por parte de la IA. 

Contador de puntuación (Best of 3/5). 

 

🛠️ Tecnologías Utilizadas 

Lenguaje: [Ej: Python 3.10] 

Control de Versiones: Git & GitHub 

Entorno: Visual Studio Code
